My Projects

  1. GSM Based Fire Alert System:
    An Arduino based project which uses LM35 Temperature Sensor to detect fire or spark presence in ambient environment. Sends a text SMS message uing SIM900 GSM Module to designated phones numbers as soon as fire threshold is overridden.

  2. Smart Home Automation:
    An Arduino based project which controls 4 channel SPDT relay module over a bluetooth network limited in a house range. Bluetooth network created by HC-05 Module setups communication bridge between arduino serial port and custom made android app.

  3. Automatic Sentry Gun:
    An Arduino based Defence project which fires a toy gun as soon as it detects a obstruction in specific radius. Toy gun is mounted on a stepper motor along with HC-SR04 ultrasonic sensor to detect obstruction. Fire Mechanism of gun is controlled by servo.
